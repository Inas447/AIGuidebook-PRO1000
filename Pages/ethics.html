Etikk og personvern i bruk av KI

Når du bruker KI i studier, handler det ikke bare om hva som er lov og ikke lov. Det handler også om ansvarlighet, transparens og respekt for personvern. Denne siden gir oversikt etiske prinsipper og hva du bør være oppmerksom på når du bruker KI-verktøy.

Hva betyr etisk bruk av KI?

Etisk bruk av KI innebærer at du:

  • • Er åpen om hvordan du bruker KI.
  • • Tar ansvar for innholdet du leverer.
  • • Ikke lar KI erstatte egen faglig innsats.
  • • Vurderer kvaliteten og troverdigheten i svarene du får.
  • • Unngår å bruke KI på en måte som kan villede andre.

Etiske prinsipper for studenter

  • • Transparens: Oppgi alltid når KI har vært brukt.
  • • Integritet: Lever kun arbeid du selv står for.
  • • Kildekritikk: KI kan ta feil - du må kontrollere fakta.
  • • Ansvarlighet: Du er ansvarlig for alt du leverer, også det KI har foreslått.

Personvern: Hva du kan og ikke kan dele med KI

Unngå å dele personlig, konfidensiell eller sensitiv informasjon med KI-verktøy.

Hva du ikke bør dele:

Å dele sensitiv informasjon med KI er risikabelt; data kan lagres, gjenbrukes eller eksponeres.

  • ✗ Personlig informasjon: adresser, telefonnumre, ID-numre, økonomiske detaljer. Dette kan føre til identitetstyveri, svindel og alvorlige personvernbrudd.
  • ✗ Konfidensiell data: private forskningsdata, upublisert arbeid og annen konfidensiell data. Å dele dette kan eksponere sensitive prosjekter, bryte avtaler eller gi uønsket tilgang til konfidensiell informasjon.
  • ✗ Andres informasjon: medstudenters data, intervjuutskrifter med identifiserbar informasjon. Å dele dette kan bryte samtykke og personvernlovgivning.
  • ✗ Innloggingsinformasjon: passord, kontodetaljer eller koder for flerfaktorautentisering. Dette kan brukes til å ta over kontoene dine og få tilgang til beskyttede data.
  • ✗ Sensitive skoledata eller eksamensbesvarelser: eksamensbesvarelser, vurderte arbeider eller konfidensiell tilbakemelding. Å dele dette kan bryte faglige integritetsregler og utsette deg eller andre for disiplinær reaksjon.

Hva du kan dele:

Du kan trygt bruke KI med anonymisert og generell informasjon som ikke kan kobles til deg eller andre.

  • • Generelle problemstillinger: for eksempel teori, fagbegreper eller metoder uten personopplysninger.
  • • Anonymiserte eksempler: fjern navn, kontaktinfo, ID-er og andre kjennetegn som kan identifisere personer.
  • • Egenprodusert utkast: tekst du selv har skrevet, når målet er språkvask, strukturhjelp eller forslag til forbedring.
  • • Offentlig tilgjengelig informasjon: kilder og data som allerede er publisert og ikke underlagt taushetsplikt.

Tenk over hva du deler med KI-verktøy: del kun det som er trygt, nødvendig og anonymisert.

Unngå brudd på opphavsrett

KI-verktøy kan i noen tilfeller gjengi opphavsrettsbeskyttet materiale. Derfor er det viktig å være oppmerksom på kildebruk og unngå plagiat, inkludert selvplagiat.

Risiko:

  • • KI-modeller er trent på store mengder internettdata, inkludert opphavsrettsbeskyttede verk. Treningsgrunnlaget kan inneholde skjevheter – ta høyde for dette i arbeidet ditt.
  • • KI-verktøy kan generere tekst eller ideer som ligger tett opptil opphavsrettsbeskyttede kilder.

Tips:

  • ✓ Omskriv KI-generert innhold med dine egne ord og vær kildekritisk.
  • ✓ Oppgi KI-verktøyet og prompten i kildelisten og henvis i løpende tekst ved direkte sitering.
  • ✓ Be KI om tilbakemelding på tekst du har skrevet selv.

Hvordan verifisere informasjon

Det er lurt å faktasjekke KI-generert informasjon før du bruker det i arbeidet ditt.

Strategier for verifisering:

1. Be KI om kilder

Be om: "Vennligst oppgi kilder for disse påstandene" eller "Hvilke studier støtter denne informasjonen?"

2. Kryssreferenser med pålitelige kilder

Sjekk fakta mot akademiske databaser, fagfellevurderte tidsskrifter eller pålitelige referansematerialer.

3. Verifiser URL-er og sitater

Åpne alltid lenker og sitater fra KI, og kontroller at de eksisterer og er korrekte.
